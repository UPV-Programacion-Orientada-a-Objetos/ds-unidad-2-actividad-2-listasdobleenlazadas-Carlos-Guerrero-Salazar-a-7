Detailed Description

Genera un objeto para gestionar la comunicación serial con un dispositivo (ej.

Arduino).

Esta clase se encarga de abrir y configurar el puerto serial, leer líneas de datos y procesarlas para convertirlas a tipos numéricos (int o float) según sea necesario, manejando el polimorfismo a través de plantillas.

bool ArduinoSerial::estaConectado ( ) const

Función plantilla para procesar y entregar un dato numérico del tipo esperado.

Verifica si el objeto ArduinoSerial ha establecido una conexión exitosa con el puerto.

Esta función llama repetidamente a leerLinea() hasta que la línea leída pueda ser parseada como el tipo de dato especificado por la plantilla (T).

  • Si T es int, solo acepta números enteros. Si T es float, acepta números con o sin decimales.
  • Template Parameters
    TTipo de dato esperado (generalmente int o float).
    Returns
    Un dato del tipo T si la lectura y conversión son exitosas. Retorna T() (valor por defecto, ej. 0 o 0.0f) en caso de fallo de lectura.
    Comprueba el estado de la conexión serial.
    Returns
    true si el puerto serial está abierto y configurado, false en caso contrario.
    true si el descriptor de archivo es válido y la configuración fue exitosa; false en caso contrario.

◆ leerLinea()

char * ArduinoSerial::leerLinea ( )

Lee una línea de datos terminada en salto de línea desde el puerto serial.

Lee una línea de datos terminada en salto de línea ('
') desde el puerto serial.

Returns
Puntero a la cadena de caracteres leída, o nullptr en caso de error o no conectado.

Esta función utiliza una lectura bloqueante (con timeout por VTIME) y almacena los bytes hasta encontrar un '
' o alcanzar el límite de 255 caracteres.

Returns
Un puntero estático a la cadena de caracteres con la línea leída, o nullptr si la conexión no está activa o si ocurre un error de lectura.
Precondition
Debe haber una conexión activa (conectado == true) con el puerto serial.
